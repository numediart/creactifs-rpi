Atelier Créactifs: Raspberry Pi
===============================

Ce répertoire héberge le matériel de cours utilisé pour les ateliers Créactifs! Raspberry Pi. Cette version de l'atelier a été donnée pour la première fois en 2026 par François Marelli (ISIA Lab, UMONS).

Pour plus d'informatiens, voir <https://le-click.be/creactifs/>
